# Secure-file-transfer-system-over-a-network
A desktop application built in Java Swing that enables secure file transfer between a client and server over a network using AES encryption and socket programming.

# 📌 About the Project
In today's digital world, secure data transmission is critical. This project implements a client-server file transfer system with end-to-end AES encryption and user authentication — ensuring that files cannot be intercepted or read by unauthorized parties during transmission.

# 🛠️ Technologies Used
- Language: Java
- UI: Java Swing (Desktop Application)
- Encryption: AES (Advanced Encryption Standard)
- Networking: Socket Programming (TCP/IP)
- Concepts: Client-Server Architecture, Authentication, Cryptography

# ✨ Features
- ✅ End-to-end AES encryption for all file transfers
- ✅ User authentication before file access
- ✅ Client-server communication via socket programming
- ✅ Desktop GUI built with Java Swing
- ✅ Secure against unauthorized interception
- ✅ A dedicated chatbot to help user

# ⚙️ How It Works
- Step 1 → Server starts and listens on a port
- Step 2 → Client connects to server
- Step 3 → User authenticates with credentials
- Step 4 → File selected by client is encrypted using AES
- Step 5 → Encrypted file is sent over the network via socket
- Step 6 → Server receives and decrypts the file
- Step 7 → File saved securely on server side

# 🔐 Security Features
| Feature | Implementation |
| :--- | :--- |
| Encryption | AES (128-bit) |
| Authentication | Username & Password |
| Protocol | TCP/IP Socket |
| Data Integrity | Encrypted payload |
